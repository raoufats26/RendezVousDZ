🎯 PRODUCT OVERVIEW
=================================================
RENDEZVOUSDZ is a smart queue management system.

Actors:
1. Business Owner (authenticated)
2. Public Client (no login)

Core Features Completed:
- Authentication system
- Business creation (1 business per owner)
- Daily queue logic (1 queue per day per business)
- Walk-in client support (owner adds manually)
- Public booking page (/b/<business_id>)
- Daily client limit enforcement
- Status management (waiting / done / skipped)
- Validation and duplicate protection
- Real-time queue updates using Flask-SocketIO
- Business settings page

=================================================

- Initializes Flask-SocketIO
- Registers blueprints
- Handles socket events:
    - connect
    - disconnect
    - join (room = business_<id>)

SocketIO runs using:

routes/auth.py:
- Login
- Register
- Session management

routes/booking.py:
Handles:
- /dashboard
- /create-business
- /settings
- /add-client (walk-in)
- /mark-done/<id>
- /mark-skipped/<id>
- /b/<business_id> (public booking)

Real-time logic:
emit_queue_update(business_id, queue_id)

'queue_updated'

🔄 REAL-TIME ARCHITECTURE
=================================================

1. Client connects via SocketIO
2. Client emits:
   'join' with business_id
3. Server joins room:
   business_<id>
4. On queue change:
   emit_queue_update()
5. Frontend updates DOM dynamically

Real-time works and must NOT be broken.
